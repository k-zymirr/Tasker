# Tasker
*A task manager from the terminal.*

## Requirements
- build-essential
- cmake

## How to install it
Run the file install.sh as root.

## Help
```
Usage: tasker [ACTION] [ARGUMENTS]

-h, --help    Display the help message
-v, --version       See the version of tasker

Actions:
    ls     List currents tasks
    add    Add a task to the list
    check  Mark a task as done
    rm     Remove a task from the list
    class  Manage classes

To see how to use an action, type tasker [ACTION] -h or --help
```
